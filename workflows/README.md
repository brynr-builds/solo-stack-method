## How to Use

1. **Identify your task** — pick the workflow that matches.
2. **Follow the steps** — each workflow is sequential.
3. **Don't skip steps** — they exist to prevent mistakes.
4. **Ask if unclear** — the Not-Psychic Rule applies.

## Key Principle

> Workflow Understanding Over Code Understanding — agents should understand *how the work flows*
> before touching code. These workflows define the "how" that makes the "what" safe.
